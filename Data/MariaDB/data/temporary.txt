微軟視窗作業系統 ( Operating System: Acer-NEO-2023 Window11 x86_64 Inter(R)-Core(TM)-m3-6Y30 )

一、自行創建自定義名爲「Database1」的資料庫：
1、啓動 MariaDB 資料庫的服務端「mariadbd」應用：
C:\> C:\Database-to-Excel-VBA\MariaDB\MariaDB10.11\bin\mariadbd.exe --defaults-extra-file=C:/Database-to-Excel-VBA/MariaDB/configuring_mariadb_with_option_files.ini --console
2、啓動 MariaDB 資料庫的用戶端「mariadb」應用：
C:\> C:\Database-to-Excel-VBA\MariaDB\MariaDB10.11\bin\mariadb.exe --defaults-extra-file=C:/Database-to-Excel-VBA/MariaDB/configuring_mariadb_with_option_files.ini --host=::1 --port=3306 --user=username --password=password Database1
3、創建名爲「Database1」的資料庫：
MariaDB [(none)]> CREATE DATABASE Database1;
4、鏈接名爲「Database1」並設定爲當前正在使用的資料庫：
MariaDB [(none)]> USE Database1;
5、在當前正在使用的資料庫「Database1」裏創建名爲「Collection1」的數據集合（數據表）並指定表結構（Schema）和字段（Field）值類型：
MariaDB [Database1]> CREATE TABLE `Collection1` (`AutoID` INT AUTO_INCREMENT PRIMARY KEY, `IntegerShort` SMALLINT NOT NULL, `IntegerLong` INT, `FloatSingle` FLOAT, `FloatDouble` DOUBLE, `TextShort` TINYTEXT NOT NULL, `TextLong` MEDIUMTEXT, `DateTime` DATETIME, `Date` DATE, `Time` TIME, `Currency` DECIMAL(10, 2) DEFAULT 0, `Image` LONGBLOB DEFAULT NULL, `Time_Stamp` TIMESTAMP DEFAULT CURRENT_TIMESTAMP);
6、查看名爲「Collection1」的數據集合（數據表）結構（Schema）：
MariaDB [Database1]> SHOW COLUMNS FROM Collection1;
7、退出 MariaDB 資料庫的用戶端「mariadb」應用：
MariaDB [Database1]> exit

二、或下載開箱即用 ( out of the box ) ( portable application ) 已配置壓縮檔「data-MariaDB10.11-Window10-AMD_FX8800P_x86_64.zip」解壓縮，將其保存至檔案夾 ( folder ) :「Database-to-Excel-VBA/MariaDB/data/」内自行安裝配置至該路徑即可.
